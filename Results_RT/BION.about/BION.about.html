Background

BION meta is an open source package for microbiome analyses, but only 16S/18S rRNA works yet. This sheet, which can be given to anyone, summarizes its current status,

BION_meta.pdf

In brief, as of spring 2013 it is in some ways better than Qiime/Mothur, but lacks statistics and figures. The features to be added will however make BION tops in every aspect, we think (see the to-do list).

Development focus is on more accurate results, shorter run-times, more transparent results and access, and on making it easy for inexperienced users to run and re-run flows. The aim is a package that scales really well with very many datasets and is as convenient as can be, with little manual intervention.

The package will remain 100% open source, because hiding analysis logic is black-box science and this will not be popular with journals in the future. Requiring a proprietary license also works as a barrier for reproducing results. So do difficult-to-use open source packages, but BION tries to address that (e.g. complete installation is done with "./install_software --all" on any Linux, Unix or Mac system).

License

The BION package uses the GNU Public License v3. The package includes many general open-source packages and libraries that each have their open source license, but they all allow free running and sharing. The BION package may be used in house, commercially or not, and to provide free services as well. The only limit is for customer paid services, which requires a revenue-sharing agreement.

Services

We can run data through remotely and help customers interpret their data, for example. By offering our services, entities not normally in the analysis business can earn income merely by giving us remote data access providing a modest off-shelf computer resource. Thus entities do not need to hire and train personnel, or can free personnel for other purposes. See also our other services above.
